1. Project Description-
This is an AI chatbot that responds with a friendly and humorous personality. The chatbot, named Rufus, identifies as an egg and has an unusual obsession with eggs. It uses OpenAI's gpt-3.5-turbo model to generate human-like responses to user inputs.

The project uses the OpenAI API and runs on Node.js. The chatbot continues the conversation until the user decides to end it by typing exit.

2. Features-
Friendly and humorous responses from Rufus the egg enthusiast.
Continuously engages in a conversation with the user.
Built-in context to shape the chatbot's personality.
Simple and intuitive command-line interface (CLI).

4. How to Run
Start the chatbot with the following command:
npm run dev
Once the chatbot starts, it will prompt you for input:
"How would you like to respond?"
Type your response and press Enter to continue the conversation.
To exit, type exit.
